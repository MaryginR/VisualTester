Что бы развернуть тестер для какого либо проекта в qt creator'е:
1. Прописываем в main ConnectApplication, как показано в примере
2. Собираем visual tester и запускаем, подключается проект для тестирования
3. Пересобираем проект из созданного CMakeList(подсказка будет в дебаге)
4. Можем писать сценарий для тестирования в main.cpp visual tester

Функционал для написания сценариев описан в main.cpp с комментариями
